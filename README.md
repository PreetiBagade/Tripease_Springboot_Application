# 🚖 TripEase - Cab Booking System (Spring Boot)

TripEase is a backend-based Cab Booking System built using Spring Boot.  
This project was developed for learning and hands-on practice of Spring Boot concepts, REST APIs, and layered architecture.

It provides REST APIs to manage customers, drivers, cabs, and cab bookings.

---

## ⚙️ Tech Stack
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL
- Maven

---

## 📌 Project Modules

### 👤 Customer Module
- Add new customer
- Get customer by ID
- Filter customers by gender
- Filter customers by gender and age
- Advanced filtering using HQL queries

### 🚗 Driver Module
- Add driver
- Get driver by ID

### 🚕 Cab Module
- Register cab under a driver

### 📅 Booking Module
- Book a cab for a customer

---

## 🏗️ Project Architecture
Controller → Service → Repository → Database

---

## 🚀 Purpose of Project
This project was built as a **learning and practice project** to understand:
- Spring Boot REST API development
- Layered architecture design
- DTO usage for request/response
- JPA and HQL queries
- Relationship handling between entities
